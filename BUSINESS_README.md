## 개요

CeoRader는 CEO와 전략 담당자를 위한 AI 기반 시장 탐지 서비스입니다. 
이 프로토타입은 실제 CEO들에게 서비스를 시연하기 위한 버전으로, 로컬 브라우저에서 동작합니다.

### 주요 특징
- **산업별 맞춤 컨텐츠**: 온볼딩 시 입력한 키워드/업종에 따라 대시보드 내용이 동적으로 변경
- **5가지 산업 템플릿**: 물류, 제조, 금융, AI, 커머스 각각에 최적화된 시나리오 제공
- **실제 움직이는 것처럼**: 로딩, 전환, 데이터 저장 등 UX가 실제 서비스와 동일

- 라우팅 AI 도입, 친환경 차량 전환 등 실질적 액션 아이템 제시

## 주요 기능 설명

### 1. 로그인 (/login)
- Google 로그인 버튼 클릭 시 즉시 로그인 (데모 계정)
- 로딩 스피너와 약간의 지연으로 실제 인증 느낌 연출
- "데모 모드" 배지로 현재 상태 표시

### 2. 온볼딩 (/onboarding)
- 5단계 진행: 전략 목표 → 회사 설명 → 키워드 → 경쟁사 → 완료
- 각 단계마다 진행률 바 표시
- 입력값에 따라 대시보드 컨텐츠가 동적으로 결정됨

### 3. 대시보드 (/ - 홈)
- **핫 토픽**: 산업별 맞춤 Top 3 토픽 표시
- **전략 인사이트**: AI가 분석한 시장 요약 및 중요도
- **실행 가능한 다음 단계**: 구체적인 액션 아이템 카드
- **시간 필터**: 7일/30일/90일 뷰 (UI만, 실제 필터링은 DB 연동 후)

### 4. 시장 신호 (/signals)
- 실시간 감지된 시장 이벤트 목록
- 카테고리 필터링: 기회/위협/트렌드
- 중요도 점수(1-100) 표시
- 더블 클릭으로 상세 보기 (향후 구현)

### 5. 금고 (/vault)
- 저장된 인사이트/액션/토픽 목록
- 기본 샘플 데이터 2개 포함
- 삭제 기능 제공
- 공유 버튼 (향후 구현)

### 6. 설정 (/config)
- **프로필 카드**: 사용자 정보 및 가입일
- **비즈니스 설정**: 온볼딩 데이터 수정 가능
- **서비스 설정**: 온볼딩 재진행, 로그아웃
- **앱 정보**: 버전 및 데모 모드 표시

## 데이터 흐름

```
[로그인] → [온볼딩] → [대시보드]
                ↓
         LocalStorage 저장
                ↓
   [Signals] ←   → [Vault] ←   → [Config]
```

**저장되는 데이터:**
- `user`: 사용자 정보 (이름, 이메일, 가입일)
- `onboardingData`: 전략 목표, 키워드, 설명, 경쟁사
- `onboarded`: 온볼딩 완료 여부
- `temp_logged_in`: 로그인 세션
- `signals`: 시장 신호 목록
- `vault`: 저장된 아이템 목록

## 산업별 템플릿

현재 5가지 산업에 대해 최적화된 컨텐츠를 제공합니다:

| 산업 | 키워드 | 토픽 예시 | 액션 예시 |
|------|--------|-----------|-----------|
| 물류 | 물류, 배송, 운송 | AI 기반 물류 최적화 | 라우팅 AI 도입 |
| 제조 | 제조, 공장, 생산 | 스마트팩토리 전환 | 예측 정비 시스템 |
| 금융 | 금융, 은행, 투자 | AI 자산 관리 | 로보어드바이저 고도화 |
| AI | AI, 인공지능, GPT | 멀티모달 AI 상용화 | On-Device AI 개발 |
| 커머스 | 커머스, 쇼핑, 이커머스 | 라이브 커머스 성장 | 실시간 추천 AI |

**매칭 로직:**
- 키워드나 회사 설명에 특정 단어가 포함되면 해당 산업으로 분류
- 매칭되지 않으면 기본(물류) 템플릿 사용

## CEO에게 어필하는 포인트

### 1. 개인화된 인사이트
- 입력한 회사 설명과 키워드를 바탕으로 "이 회사에 꼭 필요한" 내용만 표시
- 일반적인 뉴스가 아닌, 전략적 의사결정에 필요한 정보 제공

### 2. 구체적인 액션 아이템
- "AI를 도입하세요"가 아닌
- "APAC 노선 대상 LLM 기반 배차 시스템 파일럿을 6개월 내 도입하세요"와 같은 구체성
- 예상 효과(마진 12% 증가 등)를 함께 제시

### 3. 실시간성
- "3시간 전", "8시간 전" 등 freshness 강조
- 경쟁사 움직임, 규제 변화 등 시의적절한 정보

### 4. 직관적인 UI
- 모바일 퍼스트 디자인 (CEO들은 주로 모바일로 확인)
- 핵심 정보는 한 눈에, 상세는 터치 한 번으로

## 주의사항

1. **브라우저 데이터**: 모든 데이터는 브라우저 localStorage에 저장됨
   - 시크릿 모드에서는 세션이 유지되지 않음
   - 캐시 삭제 시 모든 데이터 초기화

2. **AI 분석**: 현재는 시뮬레이션 모드
   - 실제 OpenAI API 호출은 하지 않음
   - 미리 정의된 템플릿에서 컨텐츠 선택

3. **데모 계정**: 로그인 시 자동 생성
   - 이메일: ceo@company.com
   - 이름: 홍길동

4. **가치**: "월 OO만원 정도라면 구독하실 의향이 있으신가요?"

5. **부족한 점**: "어떤 정보가 더 있었으면 좋겠나요?"

## 지원 및 문의

프로토타입 시연 중 문제가 발생하면:

1. 브라우저 개발자 도서(F12) → Console 확인
2. localStorage 데이터 확인: Application → Local Storage
3. 초기화가 필요하면: localStorage.clear() 실행 후 새로고침
